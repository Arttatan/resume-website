# Иллюстрации для разделов сайта

В этой папке должны находиться изображения для каждого раздела сайта.

## Требуемые изображения:

1. **about-icon.png** - Иллюстрация для раздела "О себе"
   - Рекомендация: профиль/портрет, иконка пользователя
   - Размер: 80x80px (или больше, будет масштабироваться)

2. **skills-icon.png** - Иллюстрация для раздела "Навыки"
   - Рекомендация: галочка/чеклист, инструменты, звезды навыков
   - Размер: 80x80px

3. **experience-icon.png** - Иллюстрация для раздела "Опыт работы"
   - Рекомендация: документ/резюме, портфель, timeline
   - Размер: 80x80px

4. **education-icon.png** - Иллюстрация для раздела "Образование"
   - Рекомендация: диплом/сертификат, шапка выпускника, книга
   - Размер: 80x80px

5. **cases-icon.png** - Иллюстрация для раздела "Кейсы"
   - Рекомендация: **конверт** (✉️)
   - Размер: 80x80px

6. **contacts-icon.png** - Иллюстрация для раздела "Контакты"
   - Рекомендация: телефон/почта, сеть/коммуникация, контакты
   - Размер: 80x80px

## Формат:
- PNG с прозрачным фоном (рекомендуется)
- SVG (также поддерживается)
- JPG (если без прозрачности)

## Стиль:
- Современный, технологичный
- Подходит под тему QA/IT
- Не перегруженный деталями
- В цветовой гамме сайта (фиолетовый/синий) или нейтральный

## Fallback:
Если изображение не найдено, будет показана эмодзи-иконка как запасной вариант.
