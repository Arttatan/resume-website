# QA Engineer Resume Website

Персональный сайт-резюме QA Engineer

## Локальный просмотр

Просто откройте файл `index.html` в браузере

## Размещение в интернете

### Вариант 1: GitHub Pages (бесплатно)

1. Создайте репозиторий на GitHub
2. Загрузите файлы `index.html` и `styles.css`
3. В настройках репозитория включите GitHub Pages
4. Ваш сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория`

### Вариант 2: Netlify (самый простой)

1. Перейдите на https://netlify.com
2. Зарегистрируйтесь (можно через GitHub)
3. Перетащите папку с файлами в раздел "Deploy"
4. Сайт сразу станет доступен

### Вариант 3: Vercel

1. Перейдите на https://vercel.com
2. Зарегистрируйтесь
3. Загрузите файлы через интерфейс
4. Сайт будет размещен автоматически

## Файлы проекта

- `index.html` - основная HTML структура
- `styles.css` - стили и анимации
- `resume.md` - исходное резюме в Markdown
